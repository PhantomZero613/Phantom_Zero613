# Phantom_Zero613

This repository contains the Purple Phantom AI Assistant project.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the AI Assistant

To run the command-line version:
```bash
python Purple-Phantom.AI/Purple-Phantom.py
```

## Running the Web UI

To run the web-based UI using Streamlit:
```bash
streamlit run Purple-Phantom.AI/app.py
```

This will start a web server, and you can access the UI in your browser.

In GitHub Codespaces, the UI will be available at the provided URL (usually on port 8501).
